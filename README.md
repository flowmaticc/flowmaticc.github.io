## ¿Qué es Plantillas Macro?

Plantillas Macro es una herramienta de productividad para Windows que permite pegar plantillas de texto formateado al instante en **cualquier aplicación**: Outlook, Word, Chrome, CRM, chat de soporte, etc.

Ideal para:
- 📞 **Call centers y atención al cliente** — responde consultas recurrentes en segundos
- 🏢 **Oficinas y administración** — presupuestos, contratos, emails tipo con un solo comando
- 👩‍🏫 **Docentes online** — feedback estándar, correcciones, respuestas a alumnos
- 🩺 **Clínicas y consultas** — informes, instrucciones postoperatorias, citas
- 💻 **Soporte técnico** — respuestas a tickets, instrucciones paso a paso

---

## ¿Cómo funciona?

1. Guarda tus plantillas como archivos `.docx`, `.odt` o `.txt` en una carpeta
2. En cualquier aplicación de Windows, escribe `/` seguido del nombre de la plantilla
3. Aparece un popup de búsqueda — selecciona y pulsa Enter
4. La plantilla se pega al instante **con todo el formato original**: negritas, tablas, imágenes, estilos

```
/presupuesto  →  pega tu plantilla de presupuesto completa con formato
/saludo       →  pega tu email de bienvenida
/instrucciones →  pega las instrucciones de uso paso a paso
```
