Privacy Policy

flowmaticc
Position Caret

Last updated: · Chrome Extension v4.0.0

Short version: This extension does not collect, transmit, or store any personal data. Everything stays on your device.

What this extension does

flowmaticc - Position Caret is a companion extension for the flowmaticc desktop application. When you type your configured trigger character in a web-based text editor, the extension detects the position of your text cursor (caret) and sends those coordinates to the flowmaticc app running locally on your computer.

This allows the desktop app to display its template selector popup exactly where you are typing — without requiring any clicks or interruptions to your workflow.

Data collected

Data Purpose Storage Shared
Excluded websites list User preference — sites where the extension is disabled Local only Never
Visual indicator toggle User preference — show/hide red circle on capture Local only Never
Trigger character User preference — character that activates caret detection Local only Never
Caret coordinates (x, y) Sent to the local desktop app via localhost WebSocket only Local only Never

What we do NOT collect

  • Browsing history or visited URLs (beyond the current tab's hostname for exclusion checks)
  • Page content, text, images, or any site data
  • Personal information of any kind
  • Keystrokes or typed text (only the presence of the trigger character is detected)
  • Authentication credentials or form data
  • Device identifiers or IP addresses

Local communication

All communication between this extension and the flowmaticc desktop application happens exclusively via a WebSocket connection to localhost:8765. This is a local connection on your own machine — no data is transmitted over the internet or sent to any external server.

This is the same pattern used by other local desktop companion tools. Your data never leaves your computer.

Third parties

We do not share, sell, rent, or transfer any data to third parties under any circumstances. There are no analytics, no advertising networks, and no external services integrated into this extension.
